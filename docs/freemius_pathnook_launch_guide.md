# Pathnook Freemius Launch Guide

## Public positioning

Pathnook is software for parent-facing family learning support.
The public website and all Freemius-facing review materials should stay aligned
on these statements:

- Pathnook is software.
- Parents or guardians create and control accounts.
- Adults upload learning evidence and receive structured diagnosis output.
- Freemius handles checkout, billing portal, invoices, and recurring billing as Merchant of Record.
- Pathnook handles product access, entitlements, report history, and support.

## Required public routes

The public site should expose and keep working:

- `/`
- `/pricing`
- `/sample-report`
- `/contact`
- `/help`
- `/faq`
- `/data-deletion`
- `/legal/privacy`
- `/legal/terms`
- `/legal/refunds`

## Public billing language

Do not describe billing as "active provider" or "provider switch" in public copy.
Use Freemius-first language instead:

- Secure checkout through Freemius
- Freemius customer portal
- Freemius as Merchant of Record
- Contact Pathnook for entitlement questions or refund review

## Review checklist

- No public page should mention `FamilyEducation`.
- No public page should mention `Creem`.
- No public page should contain `will start working`, `coming soon`, or provider rollback wording.
- Pricing, billing, refund, and contact pages must tell the same story.
- Placeholder legal entity or privacy email text is acceptable temporarily, but the structure must read like a real production site.
